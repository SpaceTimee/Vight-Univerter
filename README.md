## 自我介绍:
**Vight Univerter**: 一只基于**Xamarin**框架的轻量级单位转换器

适用平台: 目前仅支持 **Android5.1+**

## 注意事项:
1. Vight Univerter仍处于开发阶段，但每个正式版发布前会尽量确保其稳定可用
2. Github Release中会保留目前能够使用的所有版本，但强烈推荐使用最新版
3. 转换结果只能与精确值大致相等，请勿将Vight Univerter用于高精度计算，
